# Sentinel Civic - Client

This is the frontend client for Sentinel Civic, built with React, Vite, and Leaflet.

## Features
- **Map & Feed**: Real-time geo-visualization of issues using React Leaflet.
- **Reporting**: Advanced media upload, audio classification, and AI-driven issue reporting.
- **Trace UI**: Real-time visualization of the backend AI agent's reasoning.

## Development

```sh
npm install
npm run dev
```

## Production Build

```sh
npm run build
```
